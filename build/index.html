Member search

GET /search

Tags: Members, Search

The Search endpoint returns information about members of the Localgiving platform. The response includes publically available information about the charities returned.

Potentially sensitive data, such as locations for organisations that work with vulnerable people, will not be returned, or will not be accurate.

Pagination is based on offset values rather than "pages".

In the API, "VCO" and member are interchangable.
